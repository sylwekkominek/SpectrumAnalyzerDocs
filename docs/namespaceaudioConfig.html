audioConfig.getData ( )

The frequency 1001.29 Hz is chosen because the sampling rate (44100 Hz) divided by the data length (4096) is approximately 10.76 Hz.

The value 1001.29 Hz is a multiple of this 10.76 Hz to minimize spectral leakage to other frequencies. This ensures that the amplitude aligns correctly with the scale in dBFS
